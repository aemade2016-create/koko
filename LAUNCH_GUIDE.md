## 💡 Quick Tips

### If Products Don't Show:
```
1. Open test-all.html
2. Click "Clear localStorage & Reload"
3. Products will load automatically
```

### If You See Errors:
```
1. Open Console (F12)
2. Check for error messages
3. Clear localStorage
4. Reload page
```

### To Reset Everything:
```javascript
// In Console (F12):
localStorage.clear()
location.reload()
```

## 🎨 Customization

### Change Colors:
Edit `css/main.css`:
```css
:root {
    --color-primary: #ec4899; /* Change this */
}
```

### Change Logo:
Edit header in each HTML file or `js/components/header.component.js`

### Add Products:
1. Login as admin
2. Go to Admin Panel
3. Click "Add Product"
4. Fill form and save

### Change Text:
Edit `js/core/i18n.js` for translations
