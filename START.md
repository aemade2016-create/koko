# 🚀 ابدأ الآن - Glow Beauty

## 📖 دليل البدء السريع

### 1️⃣ افتح الموقع
```
افتح ملف: index.html
```

### 2️⃣ جرب التسوق
1. **تصفح المنتجات** - شاهد المنتجات في الصفحة الرئيسية
2. **اضغط على منتج** - لعرض التفاصيل الكاملة
3. **أضف للسلة** - اضغط "Add to Cart"
4. **اذهب للسلة** - اضغط على أيقونة السلة في الأعلى
5. **أكمل الطلب** - اضغط "Proceed to Checkout"

### 3️⃣ سجل دخول
```
حساب الإدارة (للتجربة):
Email: admin@glowbeauty.com
Password: admin123
```

أو أنشئ حساب جديد من صفحة Login

- أو أنشئ حساب جديد
