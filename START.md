# 🚀 ابدأ هنا - دليل سريع

## الخطوة 1: تشغيل الموقع

### باستخدام VS Code (الأسهل):
```
1. افتح المجلد في VS Code
2. انقر بزر الماوس الأيمن على index.html
3. اختر "Open with Live Server"
```

### إذا لم يكن لديك Live Server:
```
1. اضغط Ctrl+Shift+X في VS Code
2. ابحث عن "Live Server"
3. اضغط Install
```

## الخطوة 4: جرب الوظائف

### ✅ أشياء يمكنك تجربتها:

1. **أضف منتج للسلة**
   - اذهب للمتجر
   - اضغط "Add to Cart"

2. **بدل الثيم**
   - اضغط على أيقونة القمر/الشمس

3. **بدل اللغة**
   - اضغط على زر "AR" أو "EN"

4. **جرب لوحة الإدارة**
   - سجل دخول كأدمن
   - اذهب لـ `admin.html`
   - أضف منتج جديد

5. **جرب الكوبونات**
   - أضف منتجات للسلة
   - استخدم: `SAVE10` أو `SAVE50` أو `FREESHIP`
