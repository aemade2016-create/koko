## 🎯 كيفية الاستخدام

### 1. فتح الموقع
```bash
# افتح index.html في المتصفح
```

### 2. تسجيل الدخول
```
حساب الإدارة:
Email: admin@glowbeauty.com
Password: admin123

أو أنشئ حساب جديد
```

### 3. التسوق
1. تصفح المنتجات من الصفحة الرئيسية أو صفحة المتجر
2. اضغط على أي منتج لعرض التفاصيل
3. أضف المنتجات إلى السلة
4. اذهب إلى السلة وأكمل الطلب

### 4. استخدام الكوبونات
```
SAVE10   - خصم 10%
SAVE50   - خصم 50 جنيه
FREESHIP - شحن مجاني
```
